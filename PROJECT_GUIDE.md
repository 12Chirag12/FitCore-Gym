# FitCore – Gym Management System

## Project Overview

FitCore is a web-based Gym Management System built using Python Flask and Firebase Firestore. The system provides a professional platform for gym owners to manage members, memberships, attendance, workout plans, diet plans, trainers, announcements, and payments. Members can log in to view their personalized information, while administrators have complete control over gym operations through a secure admin dashboard.

---

# Objectives

- Manage gym members efficiently
- Provide separate Admin and User dashboards
- Synchronize updates in real time
- Maintain member attendance
- Manage memberships
- Publish workout and diet plans
- Send announcements
- Store member information securely
- Build a responsive modern website
